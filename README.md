# Algorithms and Data Structures - Skip List

![Skip List](./media/skip-list.svg)

## Characteristics

- Time complexity - Search / Insertion / Deletion:
  - Best: Ω(1)
  - Average: Θ(log n)
  - Worst: O(n) - very rare to happen
- Space complexity: O(n log n)
- Probabilistic Data Structure
